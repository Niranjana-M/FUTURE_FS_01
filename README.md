## 🚀 Project Description

This is a responsive personal portfolio website developed using React and Vite.

The portfolio includes:

* Home Section
* About Section
* Skills Section
* Projects Section
* Contact Form with EmailJS
* Resume Download
* GitHub and LinkedIn Links

---

## 🛠️ Technologies Used

* React.js
* Vite
* CSS
* React Icons
* EmailJS
